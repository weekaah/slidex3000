# Slidex 3000
Plain JavaScript Slider

## Demo
[Demo](https://weekaah.github.io/slidex3000)

## Installation
Include `slidex3000.min.js` inside `<body>` tag.
```html
<script src="slidex3000.min.js"></script>
```

## Usage
Initialize the plugin by running `slidex3000();`

```html
<script>
  scrollprogress('body');
</script>
```
